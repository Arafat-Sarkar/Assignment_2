var let const

var let const
JavaScript uses the keyword var to declare a variable, which is function-scoped and accessible within the function it is declared with. We can access outside of '{}'. Let is a block scope that allows updating but cannot re-declare or access outside of '{}'. Const is a block scope that cannot be updated, re-declared, or accessed outside of the '{}' character.

null And undefined

null undefined
When a variable is declared but intentionally set to have 'no value'. When a variable is declared but not assigned a value, it becomes undefined.

REST API

REST API serves as a intermediary between the server and user, allowing users to access data from the server and display it in the browser.
